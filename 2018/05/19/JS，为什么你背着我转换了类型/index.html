一、if 的判断

你能完美的解答这些问题吗？

针对小括号里的表达式会转换为布尔值，但是具体转换规则是什么呢？

布尔类型为false只有以下几种情况，其余皆为true

  • undefined
  • null
  • 0 或 NaN
  • ‘’ (空字符串)
  • false

所以以上答案为： 1、2、3、5为true，4为false

二、== 的判断

使用==的时候，javascript会帮我们做类型转换，造成一些匪夷所思的结果，那么使用==的时候会在哪些情况下做类型转换，又会换成什么样子？

两个值类型相同，那么直接===进行对比，如果两个值类型不同，那么类型转换后进行比较。

转换规则如下：

  • 如果一个是null，一个是undefined，那么相等
  • 如果一个是数字，一个是字符串，先将字符串转为数字，然后比较
  • 如果一个值是true/false则将其转为1/0比较
  • 如果一个值是对象，一个是数字或字符串，则尝试使用valueOf和toString转换后比较
  • 其它就不相等了
