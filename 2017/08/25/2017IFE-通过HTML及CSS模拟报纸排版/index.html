前言

2017-百度前端技术学院编码任务：通过HTML及CSS模拟报纸排版

任务目的

  • 深入掌握CSS中的字体、背景、颜色等属性的设置
  • 进一步练习CSS布局

任务描述

  • 参考 PDS设计稿（点击下载），实现页面开发，要求实现效果与 样例（点击查看） 基本一致
  • 页面中的各字体大小，内外边距等可参看 标注图（点击查看）
  • 页面宽度固定（定宽）

实现

字体

  • font
    简写字体属性

  • font-style
    定义字体风格

  • font-variant
    定义小型大写字母的字体

  • font-weight
    定义字体的粗细

  • font-size
    定义字体的尺寸

  • font-family
    定义元素的字体系列

    浏览器会使用识别的第一个字体系列，如果没有就使用浏览器默认字体

背景

  • background
    简写背景属性

    background-size 这个属性的定义必须在 之后, 并使用 ‘/‘ 符号分隔

  • background-color
    设置背景颜色

  • background-position
    设置元素背景图片的位置

  • background-size
    设置元素背景图片的尺寸

  • background-repeat
    设置背景图片的平铺模式

  • background-origin
    background-origin 属性规定 background-position 属性相对于什么位置来定位。

  • background-clip
    设置背景的绘制区域

  • background-attachment
    设置背景图片是否固定或者随着页面滚动

  • background-image
    设置元素的背景图片

颜色


三种方式描述颜色

  • 关键字：red
  • RGB系统： rgb(255,0,51) rgba(255,0,51,.5)
  • HSL系统：hsl(240,100%,50%)或hsla(240,100%,50%,0.05)

currentColor 关键字
表示和当前元素的color一样的颜色

文本

  • text-indent
    文本缩进

  • text-align
    规定文本水平对齐方式

  • word-spacing
    字（单词）间距；

    象形文字的语言或非罗马书写体往往无法指定字间隔，注意对中文无效

  • letter-spacing
    字母间距

  • text-transform
    文本大小写

    • text-decoration
      文本的装饰
      • 一个规则中结合多种装饰
      • text-decoration属性是一种简写属性，并且可以使用普通属性三个值中的任何一个。普通属性如下：text-decoration-line ，text-decoration-color和text-decoration-style

  • white-space
    处理元素内的空白

  • direction
    属性规定文本的方向 / 书写方向

问题

  • 首行字母大写的长宽过大，布局变形
    解答：利用伪元素 first-letter 针对整体第一行的首字母进行处理
